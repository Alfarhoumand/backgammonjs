backgammon.js-model

The backgammon.js-model library, used by both backgammon.js-client and backgammon.js-server packages, provides the following functionality:

  • Object oriented model of classes modeling real-world and objects and abstract notions (model.js file);
  • Message IDs shared by client and server communication objects (comm.js file);
  • Base Rule class describing a variant of the game (rules/rule.js file);
  • Sample rules for two variants of the game: rules/RuleBgCasual.js and rules/RuleBgGulbara.js.
